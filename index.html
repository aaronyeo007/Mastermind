Objective: To break the secret code


How to play

1. Mastermind is a game of four color codes containing the colored buttons displayed.

2. Each 4 color codes may contain pegs of the same color.

3. Guess the correct code (color and order) by clicking on the colored buttons on the right.

4. Each guess will be compared with the code. For every correct color that is in the correct position, a red peg will be displayed.

5. However, for every correct color that is in the wrong position, a white peg will be displayed.

6. Red peg takes precedence over the white peg.
i.e. if the secret code contains 1 green, and the code breaker guessed 2 green with one of the green in the correct position, red peg will be displayed in preference to white peg.
