Objective: To break the secret code


How to play

1. The code is a four color code containing the colored buttons displayed.

2. The code may contain multiples of the same color.

3. Try to guess the correct code (color and order) by clicking on the colored buttons.

4. Your guess will then be compared to the code. For every correct color that is in the correct position, a red peg will be displayed.

5. However, for every correct color that is in the wrong position, a white peg will be displayed.

6. Red peg takes precedence over the white peg.
i.e. if the secret code contains 1 green, and the code breaker guessed 2 green, then the code maker gives 1 red hint peg if one of the green is in the correct position, so the code maker has used red peg in preference to white peg.
